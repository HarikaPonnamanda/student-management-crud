# Student Management CRUD (C# Console App)

This is a simple C# console-based CRUD application that allows you to:

- Add Students  
- View Students  
- Update Student Details  
- Delete Student Records  

## Tech Stack
- C#
- .NET 8 Console Application

## How to Run
1. Open the folder in Visual Studio or VS Code  
2. Run the program using **F5** or `dotnet run`
